01  ·  The problem

Nobody hands an agent the treasury keys.

An agentic economy needs treasuries that run unattended — rebalancing exposure, paying suppliers on time, buying the data they need to decide.

But an agent holding the keys can do anything. And an agent constrained only by its own instructions is constrained by nothing. One bad inference, one injected prompt, and the treasury is gone.

02 — Problem
02  ·  The thesis

The agent proposes.
The vault disposes.

Off-chain, the agent forms an intent. It can be wrong, confused or hijacked. It holds no privilege of its own.

On-chain, the vault re-checks token, payee, cap and slippage in a contract the agent does not control. It reverts or it settles. It cannot be talked out of it.

03 — Thesis
03  ·  The loop

Every fifteen minutes, unattended.

01

Read

Vault balances, the desk rate, and a live EUR/USD quote it pays for by the request.

02

Decide

Drift outside the 60/40 band? Payment due? It builds exactly one call. Nothing else is reachable.

03

Enforce

The vault re-checks every rule on-chain, then reverts or settles atomically. No admin key in the path.

What matters is the absence: no human approval step, no off-chain allowlist to trust, no “please don’t do that” buried in a system prompt.

06 — Live proof
06  ·  Agent Nanopayments

It pays for its own data.

Every FX quote is bought from an x402-priced oracle at 0.0005 USDC, settled through Circle Gateway. Machine to machine — no invoice, no seat licence, no human in the billing loop.

An agent waking 96 times a day shouldn’t buy a monthly data plan. It should pay per decision. If the paid oracle is unreachable it falls back to a free source and says so.

07 — Nanopayments
07  ·  Why Arc

Gas in USDC changes what an agent can be.

An agent that must hold a volatile gas token to move stablecoins has a second treasury problem it never asked for. On Arc it holds one asset class and spends it — including on gas.

Worth stating rather than hiding: Circle’s own spending policies are mainnet-only today. On Arc testnet the vault is the enforcement layer — precisely the thing we set out to build.
